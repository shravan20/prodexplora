<p align="center">An app in progress to make a difference and connecting two dimension.</p>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development in watch mode
$ npm run dev

# production mode
$ npm run prod:start

# production build mode
$ npm run prod:build
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
